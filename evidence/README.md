# Evidence directory

This directory contains only publication-safe material needed to audit the scoped findings.

- controlled-specimens/ contains one verified 133-byte ZIP. Its present bytes are independently verified; its asserted historical browser-download origin remains first-party attestation.
- screenshots/ contains redacted derivatives, explicit transcripts, and hashes of originals that remain withheld.
- timeline/ contains two operator-supplied exact recovered transcript extracts, each with an explicit no-export/no-screenshot ceiling.
- progress/ contains a deterministic crop of a recovered PDF-embedded false-progress/admission visual and its provenance record.
- public-records/ contains concise public-record metadata and one archived first-party status response with a safe-header allowlist.
- tables/ contains curated account-record, state-model, derived Book/rename, and date-only post-resolution tables.

The directory intentionally omits private conversation exports, unrelated manuscripts, translation packages, account identifiers, raw unredacted images, object IDs, connector data, temporary renders, and duplicated extraction trees.
