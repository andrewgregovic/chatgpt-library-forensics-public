- controlled-specimens/ contains one verified 133-byte ZIP. Its present bytes are independently verified; its asserted historical browser-download origin remains first-party attestation.
- screenshots/ contains redacted derivatives, explicit transcripts, and hashes of originals that remain withheld.
- public-records/ contains concise metadata for directly relevant public first-party records, not copied webpages.

No `timeline/` exhibit directory is present in this export. The proposed 13 July 01:34 image and 06:50 Library events remain blocked because no original direct transcript or screenshot was recovered; see the root-level article-evidence gate.
